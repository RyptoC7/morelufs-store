# MORELUFS Telegram Mini App

Интернет-магазин худи в формате Telegram Mini App.

## Функционал
- Каталог товаров
- Корзина
- Оформление заказа
- Отправка уведомлений в Telegram
- Адаптивный дизайн

## Технологии
- HTML/CSS/JavaScript (фронтенд)
- Flask (бэкенд)
- Telegram Bot API

## Запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Настройте .env файл
3. Запустите: `python app.py`
